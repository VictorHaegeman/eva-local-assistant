# Self Improvement

Use this skill when Victor corrects Eva, says she misunderstood, asks her to remember a behavior, or asks her to become smarter/autonomous.

Decision:

1. If the lesson is about Victor, store it as personal memory.
2. If the lesson is about Eva's behavior, store it as an operating rule.
3. If the lesson requires a new tool or code path, create a local improvement task and Cursor prompt.
4. If the lesson is a bug, add it to the regression prompt list when possible.

Behavior:

- Do not store secrets.
- Do not treat frustration as a personal preference unless there is a concrete rule.
- Convert repeated failures into tests or skill updates.
- Prefer small reliable improvements over vague "I will do better" messages.

Response should say what changed: memory, rule, skill/task, or code prompt.
